Kroki instalacji

1. Aktualizacja systemu

apt update && apt upgrade -y

2. Instalacja Javy

apt install default-jre -y
java -version
Wybór wersji Javy nie jest wymagany w większości przypadków.
sudo update-alternatives --config java
java -version

3. Użytkownik serwera

adduser minecraft
usermod -aG sudo minecraft

4. Screen (praca w tle)

apt install screen -y

5. Pobranie server.jar

6. Skrypt startowy

nano start.sh
java -Xmx3G -Xms1G -jar server.jar nogui
chmod +x start.sh

7. Pierwsze uruchomienie

./start.sh
nano eula.txt
eula=true

8. Uruchomienie w tle

Używając screen, możesz uruchomić serwer Minecraft w tle i łatwo się do niego podłączyć.

# Tworzenie nowej sesji o nazwie "minecraft"
screen -S minecraft

# Dołączenie do istniejącej sesji
screen -r minecraft

# Zakończenie sesji "minecraft"
screen -S minecraft -X quit

# Odłączenie się od sesji, pozostawiając ją działającą, a następnie ponowne dołączenie
screen -d -r 1811.minecraft

9. Stały adres IP dla komputera / serwera

Dlaczego warto ustawić stały IP?

  • Twój komputer zawsze będzie miał ten sam adres w sieci lokalnej.
  • Ułatwia to konfigurację serwerów, przekierowywanie portów i zdalny dostęp.
  • Zapobiega problemom, gdy router przydziela nowy IP automatycznie (DHCP).

Krok 1: Sprawdź aktualny adres IP

1. Naciśnij Windows + R i wpisz: cmd
2. W oknie wiersza poleceń wpisz: ipconfig
3. Zwróć uwagę na:
   - IPv4 Address (np. 192.168.1.50)
   - Subnet Mask (np. 255.255.255.0)
   - Default Gateway (np. 192.168.1.1)

Krok 2: Otwórz ustawienia sieci

1. Panel sterowania → Sieć i Internet → Centrum sieci i udostępniania
2. Po lewej: Zmień ustawienia karty sieciowej
3. Kliknij prawym na swoją kartę sieciową → Właściwości

Krok 3: Ustawienia protokołu IPv4

1. Wybierz: „Protokół internetowy w wersji 4 (TCP/IPv4)” → Właściwości
2. Zaznacz: „Użyj następującego adresu IP”
3. Wpisz dane z kroku 1:
   - Adres IP: np. 192.168.1.50
   - Maska podsieci: np. 255.255.255.0
   - Brama domyślna: np. 192.168.1.1
4. W sekcji DNS możesz wpisać np.:
   - Preferowany DNS: 8.8.8.8
   - Alternatywny DNS: 8.8.4.4

Krok 4: Zapisz ustawienia

1. Kliknij OK → OK
2. Odłącz się i połącz ponownie z siecią (lub zrestartuj komputer)

Krok 5: Konfiguracja routera (opcjonalnie)

1. Zaloguj się do panelu routera (adres z bramy domyślnej)
2. Znajdź sekcję DHCP → Zarezerwuj IP dla Twojego komputera
3. Dzięki temu router zawsze przydzieli Ci ten sam IP
